Mini Bar Width Proposal Card · Comparison Bar

unified-chat-interface.tsx
Formula
Bar Width % = min(100, |diff_pct|)
The bar is capped at 100% width even when |diff_pct| exceeds 100 (which can happen when the proposed price is drastically below market). The bar is red when diff_pct < 0, green when diff_pct ≥ 0.

Tone Bar Width Guest Inbox · Tone Bar

guest-inbox-wired.tsx
Fixed mapping (not computed, not a percentage of text)
sentiment = "positive" → 88% (emerald)
sentiment = "neutral" → 52% (amber)

The Tone bar width is a fixed visual signal — it does not measure a numeric probability. It is deliberately set wide for positive (optimistic hospitality tone) and narrow for cautious/negative drafts to create a strong visual cue.
